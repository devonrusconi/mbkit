# Calendar Usage

## Basic Usage

Use the "onChange" prop to get the selected date from the user.

```tsx
<Calendar onChange={date => console.log(date)} />
```

## Internationalization

Use international calendars with the "locale" prop.

```tsx
<Calendar onChange={date => console.log(date)} locale="fr-fr" value={new Date(2018, 4, 3)} />
```

## Choosing Past Dates

Here's an example of choosing a date from the past 30 days.

```tsx
<Calendar 
    onChange={date => console.log(date)}
    minDate={new Date(new Date().setDate(new Date().getDate() - 31))} 
    maxDate={new Date(new Date().setDate(new Date().getDate() - 1))} 
/>
```

## Navigate by Month

User can select the month to zoom out and navigate by month.

```tsx
<Calendar 
    onChange={date => console.log(date)}
    minDetail="year"
/>
```

## Date Ranges

Use the "selectRange" prop to support date ranges.

```tsx
<Calendar 
    onChange={date => console.log(date)}
    value={[new Date(2019, 11, 23), new Date(2019, 11, 27)]}
    selectRange={true}
/>
```

For the underlying calendar props see the <a href="https://github.com/wojtekmaj/react-calendar#readme" target="_blank">react-calendar documentation</a>.
